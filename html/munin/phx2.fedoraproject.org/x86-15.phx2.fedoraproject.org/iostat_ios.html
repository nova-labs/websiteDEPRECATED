Overview :: phx2.fedoraproject.org :: x86-15.phx2.fedoraproject.org :: iostat ios

For each applicable disk device this plugin shows the latency (or delay) for I/O operations on that disk device. The delay is in part made up of waiting for the disk to flush the data, and if data arrives at the disk faster than it can read or write it then the delay time will include the time needed for waiting in the queue.
